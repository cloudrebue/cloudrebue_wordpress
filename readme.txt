== Description ==

This plugin enables you to Send Woocomerce Notifications Via SMS and Access Bulk SMS Portal straight from the WordPress backend or via the programmers API.

All you need is the plugin and [bulk.cloudrebue.co.ke](https://bulk.cloudrebue.co.ke) account.

== Installation ==

This section describes how to install the plugin and get it working.

1. If you haven't already, then go to [bulk.cloudrebue.co.ke](https://bulk.cloudrebue.co.ke) and create a free account.
1. Install and activate the plugin.
1. Go to "Settings Â» CloudRebueSMS Settings" and add Your account details from your bulk.cloudrebue.co.ke account.
1. (Optional) Enable the SMS Portal UI and then go to "Cloud Rebue SMS Â» SMS Portal" and try to send an SMS to yourself, verifying that all is setup correctly. You will be required to login with your account username and password.
